    Network Security Investigation


    About

    • Module: CMP314
    • Title: Computer Networking 2
    • Grade: B+

    My assignment for this module was to carry out an investigation into a company network assessing the overall network security. The scenario is the previous network administrator of the client network left on bitter terms, and to the surprise of the client the previous admin did not create any network documentation which was a big concern. The objective is to test the security of the network and report the findings in the form of a Penetration Testing White Paper, create a detailed network map and a subnet table along with ports used across the entire network. If any vulnerabilities were found from the testing, countermeasures are to be given to the client.


    I found this assignment quite confusing and difficult. I felt like I was going around in circles throughout this investigation and still to this day get mixed up on certain parts of the network when remembering...it was that large and complex. I ended up despising this previous administrator as much as the client. Even though I didn't have that great of an experience with this assignment I still really enjoyed mapping this network using what I'd learned this term and other previous knowledge that came into play.

    White Paper

    When I was writing up what I'd discovered in this investigation I wanted my White Paper to be as detailed, easy to follow and understandable as possible. I spent a long time trying to make the process of uncovering the network flow when reading so it wouldn't be confusing at all. I didn't want anyone else to go crazy like myself trying to understand this network. I hope my report reflects this.
