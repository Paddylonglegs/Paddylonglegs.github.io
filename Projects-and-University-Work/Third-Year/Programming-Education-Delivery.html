    About

    • Module: CMP311
    • Title: Professional Project Development and Delivery
    • Grade: A+

    For this module, my group and I had to follow and deliver the proposal we created in term one of our third year. We had all of our roles for the project planned out and which tasks each of us were to undertake. My role in the project was one of the programmers. We began development on the project straight away the first week of term 2. I really enjoyed working on creating this web app using Python Flask. I'd never used Flask before this project and welcomed the challenge. All in all, the group I was in did a fantastic job and couldn't have asked for a better team to be placed into.


    My group for this project was: Myself, Isaac Basque-Rice, Grant Anderson, Kevin Devine, Ben Chadwick and Wojciech Chojnowski.


    We successfully delivered our project to the client. Our web app provides an easy method to submit student code in a .zip file to MOSS's plagiarism checker. Our web app is called mossguiabertay.co.uk.
